# Obsidian Ventures

**AI systems for serious operators.**

Landing page for [obsidianventures.xyz](https://obsidianventures.xyz)

---

## Stack

Single `index.html` — no build step, no framework. Pure HTML + CSS with Google Fonts (Inter) and a tiny vanilla JS scroll-animation observer.

## Design

- Background: `#0a0a0a` (obsidian black)
- Text: `#f0f0f0`
- Accent: gradient `#7c3aed` → `#d97706` (purple → gold)
- Font: Inter

## Deploy

Drop `index.html` on any static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages) and point `obsidianventures.xyz` at it.
